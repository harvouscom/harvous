# Product Overview

**Harvous** is a note-taking application with the tagline "Notes for the Lord."

## Core Features
- Personal note creation and management
- Rich text editing with Trix editor
- User authentication via Clerk
- Dashboard interface for note organization
- Public/private note visibility controls

## User Experience
- Clean, paper-inspired design aesthetic
- Card-based note display with preview content
- Protected dashboard routes requiring authentication
- Mobile-responsive interface

## Design System
The app uses a warm, paper-themed color palette with custom CSS variables for consistent theming. Design references are available in [Figma](https://www.figma.com/design/xRe02cHax8dHZml5fAoDdS/Harvous).
